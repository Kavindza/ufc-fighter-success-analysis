# UFC Fighter Success Analysis

## 📊 Project Overview
This project analyzes what factors predict success in UFC fighters using data science and machine learning techniques.

## 🔍 Research Questions
- What combination of physical and technical attributes best predicts fighter success?
- Do physical advantages (reach, height) matter more than technical skills?
- What distinct fighter archetypes exist in the UFC?

## 📁 Files in this Repository
- `UFC_Fighter_Success_Analysis.ipynb` - Main Jupyter notebook with complete analysis
- `UFC_Fighters_Statistics_Dataset.csv` - Dataset used for analysis
- `report.pdf` - Final project report (max 5 pages)

## 📈 Key Findings
- Technical skills are 3x more important than physical attributes
- Switch stance fighters have 14% higher win ratios
- Strike differential is the single best predictor of success
- Random Forest model predicts fighter success with 72% accuracy
